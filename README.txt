======== HOW TO COMPILE AND RUN THE PROGRAM in JAVA Environment ========

1. After unzipping, Place the makefile, Java file (TimeSeriesPrediction.java) 
   input file(product_distribution_training_set.txt) and JAR file(WEKA jar) in the same folder
2. TO COMPILE and RUN the program, JUST TYPE: make
3. The Source code will be compiled and start running
4. It takes about 1-2 minutes to make the predictions and to do the evaluation
5. The Prediction Results will be Automatically wtitten to ---> prediction.txt
6. The RMSE(Evaluation Results) and 'Mean RMSE' will be written to Console(STDOUT)
